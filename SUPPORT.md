# Support

LeftLevel Helix is currently a prototype. Support is community/best-effort unless a separate paid support agreement exists in the future.

## Where to ask for help

Use GitHub issues for:

- setup problems;
- documentation gaps;
- reproducible bugs;
- feature requests;
- test failures;
- questions about the prototype workflow.

Do not use public issues for suspected vulnerabilities. Use `SECURITY.md` instead.

## Before opening an issue

Please include:

- operating system;
- Python version;
- OpenSSL version;
- command you ran;
- exact error text;
- whether this is CLI, relay, app-store, documentation, or protocol related.

## Prototype limitations

Current support does not include:

- production deployments;
- protecting real sensitive communications;
- incident response;
- mobile app support;
- Discord/Slack/browser integrations;
- custom enterprise deployments.

Those may become future paid services or commercial offerings.
